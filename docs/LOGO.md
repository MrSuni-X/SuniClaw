# 🎨 SuniClaw Logo

## 设计理念

SuniClaw 的 Logo 融合了以下元素：

- 🐱 **猫咪** - 代表可爱、温暖、陪伴
- 🐾 **爪印** - 代表 SuniClaw 的名字
- 🐼 **熊猫** - 代表中文版（国宝）
- 🦅 **雄鹰** - 代表英文版（自由、力量）

## 版本

### 主 Logo 🐱

```
    🐱
   /  \
  🐾  🐾
  / \
 🐼   🦅
```

**含义：**
- 🐱 憨憨（可爱的智能助手）
- 🐾 SuniClaw（爪印）
- 🐼 中文版（熊猫代表中华）
- 🦅 英文版（雄鹰代表国际）

### 简化版

```
  🐱
 🐾🐾
```

### 文字版

**SuniClaw** = Suni（太阳/温暖）+ Claw（爪子/帮助）

## 颜色方案

| 颜色 | 代码 | 用途 |
|------|------|------|
| 猫咪粉 | `#FFB6C1` | 主色调（可爱） |
| 熊猫白 | `#F5F5F5` | 中文版 |
| 雄鹰蓝 | `#4169E1` | 英文版 |
| 爪印棕 | `#8B4513` | 点缀色 |
