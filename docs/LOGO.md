# 🎨 SuniClaw Logo

## 设计理念

SuniClaw 的 Logo 融合了以下元素：

- 🤖 **机器人/AI** - 代表智能助手
- 🌟 **星星/光芒** - 代表温暖和希望
- 🤝 **爪子/握手** - 代表帮助和陪伴（Claw 的含义）

## 版本

### 主 Logo

```
    ⭐
   /  \
  🤖  🐾
  / \
 ☀️   🌙
```

**含义：**
- 🤖 憨憨（智能助手）
- 🐾 Suni（爪印，代表陪伴）
- ⭐ 温暖的光芒
- ☀️🌙 全天候服务

### 简化版

```
  🌟
 🤖🐾
```

### 文字版

**SuniClaw** = Suni（太阳/温暖）+ Claw（爪子/帮助）

## 颜色方案

| 颜色 | 代码 | 用途 |
|------|------|------|
| 温暖橙 | `#FF6B6B` | 主色调 |

| 纯白 | `#FFFFFF` | 背景 |
